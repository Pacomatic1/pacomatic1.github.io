This site uses lots of iframes, modern Javascript, WebAssembly, and autoplaying audio. As such, this should be viewed in a modern broswer.
If possible, view this using a desktop or laptop on main mode. This is the intended experience.
If you use assistive technologies, use a phone, prefer the extra space, need to save on battery, or if main mode runs badly, you might prefer fullscreen mode. Even so, do this after trying main mode. Main mode is better, in my opinion.
