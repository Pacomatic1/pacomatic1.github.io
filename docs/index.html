This site uses lots of iframes, modern Javascript, WebAssembly, and autoplaying audio. As such, this should be viewed in a modern broswer. This site is quite data-heavy, so I recommend not using this on a metered connection.
If possible, view this using a desktop or laptop on main mode. This is the intended experience.
If you use assistive technologies, use a phone, prefer the extra space, need to save on battery, main mode runs badly, or you have slow internet, you might prefer accessible mode. Even so, do this after trying main mode. Main mode is better, in my opinion.
